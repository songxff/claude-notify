================================================================
  Claude Code 桌面通知系统  —  安装说明
================================================================

【这是什么】
  给 Claude Code 加一个 Windows 桌面通知功能。当 Claude 在后台
  运行(终端窗口不在最前)时,它会在以下时刻弹系统通知提醒你,
  点通知即可一键把终端窗口切回最前:
    · 回复完成
    · 空闲等待你输入
    · 需要你授权某个操作
  终端正在最前时不打扰、不弹通知。

【环境要求】
  · Windows 10 / 11
  · 已安装 Claude Code 命令行版,且使用默认的  用户目录\.claude  配置
  · 不需要管理员权限

【安装步骤】
  1. 把本压缩包【完整解压】到任意文件夹。
     (重要:不要在压缩包内直接双击运行,必须先解压出来)
  2. 双击解压目录里的「安装.bat」。
     (若文件名显示乱码,就是那个体积很小的 .bat 批处理文件;
      另一个 .bat 是卸载用的,别点错)
  3. 窗口出现绿色「安装完成」、并弹出一条测试通知,即为成功。
  4. 重启 Claude Code(已经开着的会话需要重新打开),功能生效。

【验证是否生效】
  让终端窗口退到后台 -> 等 Claude 回复完 / 等它要你输入 ->
  屏幕右下角弹出通知 -> 点通知或上面的「切换到终端」按钮 ->
  终端窗口被切回最前。

【安装器改了你电脑的什么】
  · 复制 hook 文件到:   用户目录\.claude\hooks\
  · 在 settings.json 里加 4 个 hook 配置
    (原 settings.json 会先自动备份为  settings.json.bak-时间戳)
  · 注册表 HKCU 下新增:claude-jump: 协议、通知应用身份
    (均为当前用户范围,不写系统级,不需要管理员)
  以上改动可重复运行安装器(用于升级),不会重复堆叠。

【卸载】
  双击「卸载.bat」,会移除上面列出的全部改动。

【可选:把通知转发到手机】
  安装后编辑   用户目录\.claude\hooks\notify-config.json
  支持 Bark / ntfy / 钉钉 / 自定义 webhook,字段说明见同目录
  「通知系统.md」第七章。

【更多技术细节】
  见安装后的:  用户目录\.claude\hooks\通知系统.md
